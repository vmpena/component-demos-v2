This is a demo of the Accordion component of ng-bootstrap.

The second example uses ng2-bootstrap along with Font Awesome for the up and down arrows displayed when an accordion is open or closed. A test that FontAweseome is working is displayed here: fa-camera-retro.

You need to make sure the styles[] array in angular-cli.json looks like this: 

 "styles": [
        "../node_modules/bootstrap/dist/css/bootstrap.css",
        "./assets/font-awesome-4.7.0/css/font-awesome.min.css",
        "styles.css"
      ]
