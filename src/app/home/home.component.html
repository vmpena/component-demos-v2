This project is for experiments with Angular 2 components such as ng-bootstrap.

Customer List

Example of using an Angular service to retrieve data from an api. There is a dedicated folder called "services" which is where the customer.service.ts is found.

Bootstrap 4

This project uses Bootstrap 4 alpha 6 as well as Font Awesome. This is because Bootstrap 4 no longer uses the glyphicons font. You can place Font Awesome icons just about anywhere with the <i> tag.
